Professional Terminal CLI Framework

A sophisticated, extensible command-line interface framework for Windows with plugin architecture, dynamic command loading, and comprehensive help system.

git clone https://github.com/seb07uk/psCLI.Tool.git

Key Features

Plugin Architecture

Dynamically load Python modules, PowerShell scripts, batch files, and executables.

ANSI Color Support

Rich, colorized terminal output with themed commands and professional styling.

Hot Reload

Reload plugins instantly without restarting the CLI application.

Installation & Setup

Requirements: Python 3.7+ (tested with 3.14.2) and Windows OS.

# Clone repository
git clone https://github.com/seb07uk/psCLI.Tool.git
cd psCLI.Tool

# Create venv
python -m venv .venv
.venv\Scripts\activate

# Run application
python cli.py

The CLI automatically creates settings in %USERPROFILE%\.polsoft\psCli\

Creating Custom Plugins

Create a new file in the plugins/ directory to extend functionalities:

from cli import command, Color

__author__ = "Your Name"
__group__ = "utility"

@command(name="hello")
def hello_cmd(*args):
    print(f"{Color.GREEN}Hello from plugin!{Color.RESET}")
